# MovieTime

## 🌍 About the app

Web of movies and series, with login and registration in firestore to allow the user to keep track of everything.

## 🔧 Run it yourself

Install dependencies:

```bash
npm install
```

Then run:

```bash
npm run dev
```

## Stack

- React
- Vite
- Tailwind
- Firestore
